# 컴퓨터공학 및 정보보안 용어 정리

포트폴리오 PDF에서 추출된 컴퓨터공학 및 정보보안 관련 용어들을 정리한 문서입니다.

## 📊 통계

- **총 용어 수**: 20개
- **가장 많이 나타난 용어**: 보안 (35회)
- **다음으로 많은 용어**: 알고리즘 (19회)

## 📁 포함된 파일

### 1. **cs_infosec_terms.md** (마크다운)
- 클래식 마크다운 형식
- GitHub, 블로그, 위키에서 바로 사용 가능
- 용어별 빈도와 설명 포함

### 2. **cs_infosec_terms.json** (JSON)
- 구조화된 데이터 형식
- API 연동이나 프로그래밍에 활용 가능
- 모든 메타데이터 포함

### 3. **cs_infosec_terms.html** (HTML)
- Notion 호환 형식
- 브라우저에서 직접 확인 가능
- 아름다운 스타일링 포함

### 4. **cs_infosec_terms.csv** (CSV)
- 스프레드시트 호환 형식
- Excel, Google Sheets 등에서 바로 열기 가능
- 데이터 분석에 유용

### 5. **notion_import.py** (Python)
- Notion API 연동 스크립트
- 자동으로 Notion 데이터베이스에 용어 추가

## 🔑 주요 용어

### 보안 관련 (상위 3개)
1. **보안** (35회) - 컴퓨터 시스템과 정보를 비인가 접근이나 손상으로부터 보호하는 것
2. **해킹** (16회) - 컴퓨터 시스템에 비인가 접근을 시도하는 행위
3. **취약점** (11회) - 보안 시스템의 약점으로 악용될 수 있는 부분

### 개발 관련 (상위 3개)
1. **알고리즘** (19회) - 문제 해결을 위해 컴퓨터가 수행할 일련의 단계적 절차
2. **프로그래밍** (10회) - 컴퓨터에게 특정 작업을 수행하도록 지시하는 명령어들을 작성하는 과정
3. **암호** (9회) - 정보를 비밀로 유지하기 위해 사용되는 코드화된 메시지

## 🚀 Notion 연동 방법

1. **Notion API 키 발급**
   - https://www.notion.so/my-integrations 방문
   - 새 통합 생성

2. **스크립트 실행**
   ```bash
   # 필수 패키지 설치
   pip install notion-client

   # 환경 변수 설정
   export NOTION_API_KEY="your_api_key_here"
   export NOTION_DATABASE_ID="your_database_id_here"

   # 스크립트 실행
   python notion_import.py
   ```

3. **Notion에서 확인**
   - 지정한 데이터베이스에 20개의 용어가 자동 추가됨

3. 다음 파일들 재생성:
   ```bash
   python generate_all_formats.py
   ```
